# 1354번: 무한 수열 2

**시간 제한:** 10 초 | **메모리 제한:** 512 MB

## 문제

무한 수열 A는 다음과 같다.

* Ai = 1 (i ≤ 0)
* Ai = A⌊i/P⌋-X + A⌊i/Q⌋-Y (i ≥ 1)

N, P, Q, X, Y가 주어질 때, AN을 구하는 프로그램을 작성하시오.

## 입력

첫째 줄에 5개의 정수 N, P, Q, X, Y가 주어진다.

## 출력

첫째 줄에 AN을 출력한다.

## 예제 입력 1

```
10000000 2 3 10000000 10000000
```

## 예제 출력 1

```
2
```

## 예제 입력 2

```
12 2 3 1 0
```

## 예제 출력 2

```
8
```

## 예제 입력 3

```
0 2 2 0 0
```

## 예제 출력 3

```
1
```

## 예제 입력 4

```
123 45 67 8 9
```

## 예제 출력 4

```
2
```
