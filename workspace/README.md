# C-Workspace
> Personal Dev Container-based Environment for C/C++ Development

이 저장소는 **The Iron Fortress** 인프라 내에서 운영되는 개인용 C/C++ 개발 워크스페이스입니다. Docker 컨테이너 환경에서 일관된 빌드 및 실행 환경을 보장합니다.

### 🛠️ Environment
- **Base Image**: Node.js 기반 커스텀 데브 컨테이너
- **Compiler**: `gcc` / `g++`
- **Focus**: 
  - Algorithm Problem Solving
  - C/C++, System Programming & Architecture Study

### 📂 Directory Structure
- `/ps`: 백준(Baekjoon) 및 알고리즘 문제 풀이 소스 코드
- `.vscode`: 컨테이너 최적화 에디터 설정
- `test.cpp`: 로컬 테스트용 샌드박스 파일
